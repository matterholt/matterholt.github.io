Building a Personal Blog

I feel as everyone should have a blog-type platform. Especially if there is an interest to learn about a thing. I have spent the past previous years documenting on pen and paper only to have the info lost. Then switch to Notion which is great and I will continue to document but it lacks a public face and can get messy (at least for me). So I think it is time to build a blog using a blog.A personal blog is a pretty simple, there are not too many moving parts but is how we are going to execute the parts.

What is the desired build

Before we get started building on the blog. It would be good to get an idea on how we want exactly we want to do with the blog. If we want to start putting content out probably something like Medium or DEV.TO would work. Just signing up will allow you to posting and there is no need to build anything. By far the easiest avenue, but it leave me unsatisfied. I want to be a good Front-end Developer and I want to make it my own and I want to create more a note book for my self.

So the personal blog, allow me to build something which satisfies my passion to learn. Give me place for me to document my journey. Allows me to take the freedom to style how I want and use the technology that I prefer. With all the freedom does cost something, with everything possible then it could be difficult to start. So lets break down into sections and start limiting our possibilities
